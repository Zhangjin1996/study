// 如果参数是一个原始值，或者是一个不具有then方法的对象，则Promise.resolve()返回一个新的Promise对象 -->

Promise.resolve等价于下面的写法
```js
   Promise.resolve('foo')
   // 等价于
   new Promise(resolve => resolve('foo'))
```
